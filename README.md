# 🤖 AI Melli

**هوش مصنوعی ملی — تولید تصویر و متن رایگان با GitHub Actions**

از قدرت GitHub Actions استفاده کنید و بدون نیاز به کلید API یا سرور، تصاویر و متن‌های حرفه‌ای با هوش مصنوعی بسازید. تمام پردازش‌ها روی سرورهای رایگان GitHub انجام می‌شه.

---

## ✨ قابلیت‌ها

| قابلیت | توضیح |
|--------|-------|
| 🎨 **تولید تصویر** | ساخت تصویر با ۸ مدل مختلف AI |
| 📝 **تولید متن** | تولید متن با ۴ مدل زبانی |
| 🧠 **تصویر هوشمند** | بهبود خودکار پرامپت + تولید تصویر |
| 🖼️ **گالری HTML** | صفحه گالری زیبا با طراحی dark theme |
| 📦 **آپلود خودکار** | ذخیره در ریپازیتوری یا GitHub Releases |
| 💰 **کاملاً رایگان** | بدون نیاز به API Key یا سرور |

---

## 🚀 نحوه استفاده

### ۱. ورک‌فلوها رو کپی کنید

فایل‌های زیر رو در پوشه `.github/workflows/` ریپازیتوری‌تون قرار بدید:

```
.github/workflows/
├── image-generation.yml          ← تولید تصویر
├── text-generation.yml           ← تولید متن
└── smart-image-generation.yml    ← تصویر هوشمند
```

### ۲. از تب Actions اجرا کنید

1. به ریپازیتوری برید
2. تب **Actions** رو بزنید
3. از لیست سمت چپ ورک‌فلو مورد نظر رو انتخاب کنید
4. دکمه **Run workflow** رو بزنید
5. مقادیر رو وارد کنید و **Run** کنید

## 📋 ورک‌فلوها

### 🎨 Image Generation

تولید مستقیم تصویر با AI:

| ورودی | توضیح | پیش‌فرض |
|-------|-------|---------|
| **prompt** | پرامپت (انگلیسی بهتره) | — |
| **negative_prompt** | چه چیزهایی نباشن | خالی |
| **model** | مدل AI | `flux` |
| **width** | عرض (پیکسل) | `1024` |
| **height** | ارتفاع (پیکسل) | `1024` |
| **num_images** | تعداد تصاویر (۱-۱۰) | `1` |
| **seed** | سید (خالی = تصادفی) | خالی |
| **nologo** | حذف واترمارک | `true` |
| **upload_method** | ریپازیتوری یا ریلیز | `repository` |

### 📝 Text Generation

تولید متن با AI:

| ورودی | توضیح | پیش‌فرض |
|-------|-------|---------|
| **prompt** | پرامپت | — |
| **model** | مدل AI | `openai` |
| **temperature** | خلاقیت (۰ تا ۱.۵) | `0.8` |
| **system_message** | نقش AI | راهنمای عمومی |
| **upload_method** | ریپازیتوری یا ریلیز | `repository` |

### 🧠 Smart Image Generation

اول پرامپت رو با AI بهینه می‌کنه، بعد تصویر می‌سازه:

- پرامپت رو به هر زبانی بنویسید (فارسی، انگلیسی، ...)
- AI خودش ترجمه و بهینه می‌کنه
- تصاویر با کیفیت بالاتر تولید می‌شه
